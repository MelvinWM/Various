Basic circle-fractal drawing - pure JavaScript variant

Note: This page requires that several features (such as JavaScript web workers) are supported by the used browser.

Warning: The "canvas" element is not supported by this browser.

Features:

  • Drawing circle-fractal.
  • Various settings and minor validation.
  • Progress status in case drawing takes a long time (try iterations = 100).
  • Cancelling drawing in progress and draw something different.
